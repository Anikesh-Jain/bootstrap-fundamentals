# Bootstrap Fundamentals

A practice repository focused on learning Bootstrap fundamentals, responsive layouts, and commonly used UI components.

## Topics Covered

### Grid System
- Basic grid layouts
- Nested columns
- Offsetting columns
- Column regrouping
- Layout shortcuts

### Forms
- Bootstrap form design
- Input styling
- Form structure and responsiveness

### Components
- Tables
- Bootstrap utility classes
- Common UI component practice

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5

## Purpose

This repository was created to practice Bootstrap fundamentals and understand how responsive frontend layouts and reusable UI components are built using Bootstrap.

## Learning Outcomes

Through this repository, I practiced:

- Bootstrap grid system
- Responsive design principles
- Form styling
- Component usage
- Layout structuring
- Utility classes
- Frontend UI organization
